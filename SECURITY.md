# 安全策略（草案）

## 支持的版本

本节说明项目的哪些版本目前会获得安全更新支持。

| 版本    | 支持状态           |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark:（支持） |
| 5.0.x   | :x:（不支持）               |
| 4.0.x   | :white_check_mark:（支持） |
| < 4.0   | :x:（不支持）               |

## 报告漏洞

本节说明如何报告项目中存在的漏洞。

请告知报告渠道、报告后可多久收到漏洞相关更新，以及漏洞被采纳或拒绝时的后续处理方式等信息。
